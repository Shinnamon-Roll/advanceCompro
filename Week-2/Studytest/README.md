# Studytest — Week-2

ไฟล์โจทย์มี 6 ข้อ อยู่ในโฟลเดอร์นี้ (ข้อละไฟล์):
- `problem1_countVowels.js` — นับสระ
- `problem2_greet.js` — template literal
- `problem3_process.js` — arrow + array
- `problem4_censor.js` — แก้คำหยาบ
- `problem5_fibGenerator.js` — generator ฟิโบนาเชีย
- `problem6_analyzeText.js` — โจทย์รวม

วิธีใช้งานอย่างเร็ว:
- เปิดเทอร์มินัลในโฟลเดอร์โปรเจค แล้วรันไฟล์ด้วย Node.js เช่น:

```bash
cd Week-2/Studytest
node problem1_countVowels.js
```

การทำงาน:
- แต่ละไฟล์มีคำอธิบาย, hint และ stub (TODO) ให้เติมโค้ด
- ถ้าต้องการเฉลยหรือให้ผมเติมโค้ดตัวอย่างให้ทั้งหมด บอกว่า "ขอเฉลยทั้งหมด" หรือขอเฉลยทีละข้อ (ตัวอย่าง: "ขอเฉลยข้อ 1")

ต้องการให้ผมทำอะไรต่อ:
- เติมเฉลยครบทุกไฟล์
- เติมเฉลยทีละข้อ ตามที่คุณขอ
- เพิ่มไฟล์ทดสอบรวม (runner) สำหรับรันทุกข้อพร้อมกัน

บอกผมว่าจะให้ทำต่อแบบไหนครับ
